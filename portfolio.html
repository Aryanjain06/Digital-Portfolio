Semester 4 - Project

Project Title - Library Management System

  • Winter Semester 2020-2021

• While seeking for a book in the university’s library, students must check what category the book belongs to and check the books within this category one by one. Students also might forget to return the book. Usually, it has to roll out fine but complex tasks to maintain records of the books.
• We developed a software to manage manual functions of a library. The software helps to manage the entire library operations from maintaining book records to issue a book. In addition, it allows streamlined management of fine details of books such as author name, edition, and many other important details. So, it is easier to search for books and find the right materials for students and the librarian.
• It was a group project with a team of 4 members.
Technology we used - MySQL, Wamp-Server, PHP, CSS and Javascript.
My Contribution - Application design, Various Diagrams, Database tables, coding for several functions(admin & student login, Barcode Scanner, Question Paper), research work and presentation.

Semester 5 & 6 - Project

Title - Human Healthcare Volunteering and Monitoring System

  • Fall Semester 2021-2022 - Winter Semester 2021-2022

• The main objective was to detect Heart failure that goes unnoticed or simply we call it as a silent heart attack by some measures like AI-based biomarker sensor using saliva and heart rate.
• The app facility for the user that makes it more simple and informative, It takes all required inputs from the user like their saliva heart beat through the hardware or you can enter it mannualy. Then hardware processes these data as desired using biomarker sensors and then send those resultant measured outputs to the web/app. Then the AI model analyses these output data on the basis of thresholds set in the software for the different types of measured quantities. Then the output will be provided on the screen.
• It was a group project with a team of 8 members.
Technology we used - AI model and android studio with java and firebase for mobile backend as a service.
My Contribution - Application design, Various Diagrams, Database tables, coding for Android APK, research work and presentation.
• We also checked the accuracy rate using Decision Tree which is the most powerful and popular tool for classification and prediction, which gave us an accuracy of 79%. And also checked using Linear Regression which gave us an accuracy of 90%.

  • January 2022

• My main objective behind this initiative Project ”StrayCare” is to Protect and Care for Stray animals. We will connect with various NGO’s to work together and save many animals.
